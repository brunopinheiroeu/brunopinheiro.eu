# Strapi Cloud (legado)

O backend Strapi não é mais utilizado pela versão atual do portfólio. Todo o
conteúdo vem do Contentful. Este arquivo foi mantido apenas como referência
histórica.

- Para integrar novos conteúdos, siga o guia
  [`CONTENTFUL_SETUP.md`](./CONTENTFUL_SETUP.md).
- Para configurar variáveis de ambiente e deploy, consulte
  [`VERCEL_DEPLOY.md`](./VERCEL_DEPLOY.md).

Se for realmente necessário religar o Strapi local, utilize a pasta
`../backend`, mas lembre-se de que o frontend não usa mais os endpoints desse
servidor.
